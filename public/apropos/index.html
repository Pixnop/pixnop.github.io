🎯 Mon parcours
#

Étudiant en BUT Informatique à l’IUT de Montpellier, j’ai découvert ma passion pour le développement web dès mes premiers cours de programmation. Cette formation m’a permis d’acquérir des bases solides en algorithmique, conception de systèmes et développement d’applications.

Formation actuelle : 3ème année de BUT Informatique - Parcours Développement d’Applications

Au-delà du cadre académique, je développe constamment mes compétences à travers des projets personnels et des défis techniques. Cette approche autodidacte me permet de rester à la pointe des technologies web modernes.

💻 Ma philosophie de développement
#

  1. Centré sur l'utilisateur

    Priorité #1

    Chaque fonctionnalité, chaque interface est pensée pour offrir la meilleure expérience possible. Je crois fermement qu'un bon développeur doit d'abord comprendre les besoins réels des utilisateurs.

  2. Code de qualité

    Excellence

    J'applique les meilleures pratiques : code propre, tests automatisés, documentation claire. Un code maintenable aujourd'hui, c'est du temps gagné demain.

  3. Performance optimale

    Rapidité

    Optimisation des ressources, lazy loading, cache stratégique... Je m'assure que chaque application soit rapide et réactive, même sur des connexions limitées.

  4. Innovation constante

    Créativité

    Veille technologique active, exploration de nouvelles approches, remise en question des solutions existantes. L'innovation guide chacun de mes projets.
